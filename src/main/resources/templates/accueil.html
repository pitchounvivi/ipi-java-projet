Bienvenue dans votre librairie audio !

Cette application web est paramétrée pour communiquer avec une API REST accessible à l'adresse http://localhost:5366. Vous trouverez ici le fichier database.sql qu'il est nécessaire d'exécuter sur votre serveur MySQL. Vous devez créer le projet Java from scratch. Pour cela utilisez https://start.spring.io/ et regardez le schéma SQL pour créer les entités Java correspondantes.

Il est nécessaire de développer les services webs nécessaires pour que cette application fonctionne. Voici l'ensemble des fonctionnalités :

  • 1 - Afficher un artiste

    En cliquant ici, on peut afficher les informations basiques de l'artiste d'identifiant 1 (AC/DC). L'appel qui est effectué est GET /artists/1. En cliquant ici, on essaye d'afficher l'artiste d'identifiant 0 mais on doit obtenir une erreur 404 car il n'existe pas.

  • 2 - Recherche par nom

    Lorsqu'on recherche le nom aerosmith dans la barre de recherche, on tombe bien sur Aerosmith ainsi que sur Aerosmith & Sierra Leone's Refugee Allstars. L'appel qui est effectué est GET /artists?name=aerosmith. Lorsqu'on recherche un artiste inexistant commme ABCDEF, on obtient une liste vide.

  • 3 - Liste des artistes

    En cliquant ici, tous les artistes sont affichés, de manière paginée. Il est possible de changer de page en utilisant les boutons. L'appel effectué est GET /artists?page=0&size=10&sortProperty=name&sortDirection=ASC

  • 4 - Création d'un artiste

    En cliquant ici ou via le bouton Nouvel artiste, présent dans la liste des artistes, on accède au formulaire de création d'un artiste. L'appel qui est effectué est POST /artists avec les données de l'artiste en JSON dans le champ data de la requête. Créer un artiste qui existe déjà (même nom) lance une erreur 409.

  • 5 - Modification d'un artiste

    En cliquant ici ou en consultant les détails de l'artiste Alanis Morissette (id 4), il est possible de modifier les informations de l'artiste d'identifiant 4 qui sont persistées en base de donnée lorsqu'on clique sur le bouton Enregistrer. L'appel qui est effectué est PUT /artists/4 avec les données de l'artiste en JSON dans le champ data de la requête.

  • 6 - Suppression d'un artiste

    En cliquant ici ou en consultant les détails de l'artiste Alice In Chains (id 5), il est possible de supprimer ce dernier lorsqu'on clique sur le bouton Supprimer. L'appel qui est effectué est DELETE /artists/5.

  • 7 - Ajout d'un album

    En cliquant ici ou en consultant les détails de l'artiste Alice In Chains (id 5), il est possible d'ajouter un album en renseignant le champ et en cliquant sur le . L'appel qui est effectué est POST /albums.

  • 8 - Suppression d'un album

    En cliquant ici ou en consultant les détails de l'artiste Alice In Chains (id 5), il est possible de supprimer un album en cliquant sur le . L'appel qui est effectué est DELETE /albums/X avec X correspondant à l'identifiant technique de l'album.
